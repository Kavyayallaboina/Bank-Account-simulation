# Bank Account Simulation

This is a basic Java program that simulates a bank account.  
It allows the user to deposit money, withdraw money, check balance, and view transaction history.

## Features
- Create an account with initial balance
- Deposit amount
- Withdraw amount
- Check current balance
- View transaction history
